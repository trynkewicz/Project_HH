Описание основных классов

Vacancy
хранит данные о вакансии;
валидирует зарплату;
умеет сравниваться с другими объектами по зарплате.

ApiBase и HhApi
базовый абстрактный класс API;
наследник HhApi делает запросы к hh.ru.
StorageBase и JsonStorage
абстракция хранилища;
реализация на JSON-файле.

utils.py
фильтрация вакансий по ключевым словам,
выбор топ вакансий по зарплате,
преобразование словарей в объекты.
